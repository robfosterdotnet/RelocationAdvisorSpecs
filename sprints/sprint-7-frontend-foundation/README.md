# Sprint 7: Frontend Foundation

**Duration:** 5-6 days
**Sprint Goal:** Set up React application and build core page structure

---

## Sprint Overview

Time to build the user interface! You'll learn:

- React with TypeScript
- Project setup with Create React App or Vite
- Component architecture
- React Router for navigation
- API integration with fetch/axios
- Basic styling with CSS/Tailwind

---

## What You're Building

### Pages
1. **Home** - Landing page with search
2. **Explore** - Browse/search locations
3. **Location Detail** - Single location view
4. **Rankings** - Personalized rankings
5. **Compare** - Side-by-side comparison

### Core Components
- Navigation header
- Search bar
- Location card
- Loading/error states

---

## Sprint Deliverables

- [ ] React app initialized with TypeScript
- [ ] Routing setup with all pages
- [ ] API client service
- [ ] Basic component library
- [ ] Home and Explore pages functional
- [ ] Journal entry for Sprint 7
