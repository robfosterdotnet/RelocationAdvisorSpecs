# Requirements Document - Sprint 7

---

## Map Requirements (FR-7.1)
- Display US map with Leaflet
- Markers for locations
- Popup on marker click
- Navigate to detail on popup click
- Handle marker density (clustering)

## Chart Requirements (FR-7.2)
- Bar chart for score breakdown
- Radar chart for comparisons
- Responsive sizing
- Clear labels and legends

## Rankings Page (FR-7.3)
- Weight sliders (0-100 for each category)
- Sliders must sum to 100
- Climate preference selector
- State filter (optional)
- Submit to get rankings
- Display ranked results

## Comparison Page (FR-7.4)
- Select 2-4 locations
- Side-by-side data display
- Radar chart overlay
- Clear/remove locations

## Location Detail (FR-7.5)
- All location data displayed
- Score breakdown chart
- Position vs. national average
- Links to compare/rankings
