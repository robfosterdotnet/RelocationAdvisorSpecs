# Sprint 7: Frontend Features & Visualization

**Duration:** 6-7 days
**Sprint Goal:** Build advanced features - maps, charts, rankings, and comparison views

---

## Sprint Overview

Now we add the "wow factor" - interactive maps, data visualizations, and the core ranking experience.

### Features to Build
1. **Interactive Map** - Leaflet map showing locations
2. **Data Charts** - Bar/radar charts for comparisons
3. **Rankings Page** - Preference sliders, ranked results
4. **Compare View** - Side-by-side location comparison
5. **Location Detail** - Complete with charts

---

## Sprint Deliverables

- [ ] Map component with location markers
- [ ] Chart components (Chart.js or Recharts)
- [ ] Rankings page with preference controls
- [ ] Comparison page working
- [ ] Location detail complete
- [ ] Journal entry for Sprint 7
