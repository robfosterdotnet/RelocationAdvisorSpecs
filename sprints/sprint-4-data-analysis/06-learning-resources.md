## Data Analysis Concepts

### Descriptive Statistics
- Mean, median, mode
- Standard deviation
- Percentiles and quartiles
- Distributions (normal, skewed)

### Data Quality
- Completeness - Is data missing?
- Accuracy - Is data correct?
- Consistency - Is data uniform?
- Timeliness - Is data current?

### Exploratory Data Analysis (EDA)
- Understanding data distributions
- Finding outliers
- Identifying correlations
- Discovering patterns

## Key Concepts for This Sprint

1. **Business Intelligence (BI)** - Using data to drive business decisions
2. **Data Modeling** - Structuring data for analysis (star schema, relationships)
3. **ETL vs. BI** - You did ETL (Extract-Transform-Load), now you're doing BI (analysis)
4. **Measures vs. Calculated Columns** - When to use each in Power BI
5. **Cross-Filtering** - How visuals interact with each other
6. **Data Storytelling** - Presenting insights effectively
